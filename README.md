# AI-POWERED-CUSTOMER-CHURN-PREDICTION
AI-Powered Customer Churn Prediction System  Built an end-to-end machine learning pipeline to predict customer churn using Random Forest. Performed data cleaning, feature engineering, EDA, and model evaluation on 7,000+ customer records. 
# Customer Churn Prediction

## Objective
Predict customer churn using machine learning.

## Technologies
- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit

## Workflow
1. Data Cleaning
2. EDA
3. Feature Engineering
4. Model Training
5. Evaluation
6. Deployment

## Accuracy
~80% using Random Forest
